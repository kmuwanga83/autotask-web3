# Autotask Web3

Autotask Web3 is a decentralized application (dApp) for automating tasks using blockchain technology with Solana wallet integration.

## 🚀 Features

- 🔐 Phantom Wallet Integration
- 📋 Task Automation
- ⚡ Built with Next.js
- 🌐 Web3 Enabled

## 🛠️ Tech Stack

- Next.js
- TypeScript
- Solana Web3.js
- Wallet Adapter

## 📦 Setup

```bash
git clone https://github.com/YOUR_USERNAME/autotask-web3.git
cd autotask-web3/app
npm install
npm run dev
