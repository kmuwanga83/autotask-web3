# Autotask Web3

Autotask Web3 is a decentralized application (dApp) designed to automate and manage tasks using blockchain technology. It integrates Solana wallet support for secure user interactions.

## 🚀 Features

- 🔐 Wallet Integration (Phantom)
- 📋 Task Automation
- ⚡ Built with Next.js
- 🌐 Web3 Enabled

## 🛠️ Tech Stack

- Next.js (Frontend)
- TypeScript
- Solana Web3.js
- Wallet Adapter

## 📦 Installation

```bash
git clone https://github.com/YOUR_USERNAME/autotask-web3.git
cd autotask-web3/app
npm install
npm run dev
